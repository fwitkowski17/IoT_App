Aplikacja na laboratorium z przedmiotu "Technologie webowe w aplikacjach internetu rzeczy"

## API
### Uruchamianie
> npm run dev

### Endpointy
* **GET** /api/data/{id}/latest
* **GET** /api/data/{id}
* **GET** /api/data/{id}/{num}
* **POST** /api/data/{id}
* **DELETE** /api/data/all
* **DELETE** /api/data/{id}
